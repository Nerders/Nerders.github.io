Java基础之结构

一、switch分支结构

语法：

switch(表达式){
        case 数值1：    // 代码1；
            break;
        case 数值2：    // 代码2；
            break;
        default:
            // 代码；
}

执行流程：

1. 如果表达式的值，等于case后的某个数值，则从该case的代码开始执行；
    如果该case后没有break,则该case下的所有代码也会执行。
2. default： 当表达式的值没有任何case的值与之匹配时，则执行default后的代码
3. default与case的顺序无碍，一般放在最后。

在JDK1.7之后，表达式的值和case后的值，继byte、short、int、char之后，支持String字符串类型

二、while结构

语法：
    while(布尔表达式){    // 布尔表达式为循环条件
        // 代码-->会被执行0-n次
        // 循环变量调整
    }
示例：
    int i = 1,sum = 0;
    while(i<=100){
        sum += i;
        i++;
    }
    System.out.println("前一百的累加和为："+sum);

三、do…while 结构

语法：
    do{
        // 代码-->会被执行1-n次
        // 循环变量调整
    }while(布尔表达式);    // 布尔表达式为循环条件
示例：
    int i = 1,sum = 0;
    do{
        sum += i;
        i++;
    }while(i<=100);
    System.out.println("前一百的累加和为："+sum);
注意：
    即使循环条件不成立，do{ }中的代码块也会被执行一次。
    上面代码中，即使循环不成立，i的值也会+1

四、for循环

语法：
    for(循环变量初始化；循环条件；循环变量调整){
        // 循环体（会被执行0-n次）
    }
示例：
    int sum = 0;
    for(int a = 1;a<=100;a++){
        sum += a;
    }
    System.out.println("前一百的累加和为："+sum);

    for (int j = 1; j <= i; j++) {

五、总结

  1. 循环结构的特点：

    do..while:    循环体执行 1-n 次
while:    循环体执行 0-n 次
for:    循环体执行 0-n 次

  2. 循环结构的使用场景：

    do...while:    不确定循环次数时
while:    不确定循环次数时
for:    确定循环次数时

  3. continue与break的区别：

    1> break跳出当前循环
2> continue跳出本次循环
3> break可以使用在switch结构中，而continue不行
