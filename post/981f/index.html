Java基础

一、名词解释

1）JavaSE java标准版开发平台
2）JavaME java移动端开发平台
3）JavaEE java企业级开发平台

JVM: Java虚拟机 屏蔽不同操作系统间的底层差异
JRE: Java运行环境 JRE = JVM+解释器
JDK: Java开发者工具包 JDK = JRE+编译器+类库+工具

二、运行机制

1. 编译型： 源代码  -->  编译器编译  -->  机器码
    优点：效率高
    缺点：难以跨平台
    代表性语言：C系列语言

2. 解释型： 源代码  -->  在执行时，被一行行的解释为机器码（解释器）即时翻译
    优点： 可以跨平台
    缺点； 效率低
    代表性语言： HTML

3. Java的运行机制： 先编译、后解释运行
    (.java)源文件  -->  编译  -->  (.class)字节码文件  -->  将字节码文件逐行解释并运行

三、Java程序

public class 类名{    //class: 类，代码的容器
    public static void main(String[] args){    //主函数
        System.out.println("Hello World");
    }
}  

要点：
1. 在一个类中，有且只能有一个main（主）函数
2. 在一个源文件中，可以写多个平行class（类），并且每个class都会在编译后，生成相对应的.class文件
3. 在一个类中的多个class的类名不可以重复

公开类： 如果一个类被public所修饰，那么该类为公开类
1. 一个源文件中只能有一个公开类
2. 声明了公开类的源文件的文件名必须与公开类类名一致
package: 包,用于管理.class文件
语法： 
    package 一级包名.二级包名.三级包名...;
注意：
    1. package 语句必须在源文件的首行
    2. 一个源文件中只能有一个package语句

import: 导包
语法：
    import 包名1.包名2.类名；    // 只导入包中的一个类
    import 包名1.包名2.*；    // 导入包中的所有类
注意：
    1. import 语句在package语句以下，一般在第二行
    2. 一个源文件中可以有多个import语句
       例；
        package com.nerder;
        import java.util.Scanner;
另，java会为每个文件自动补上    import java.lang.*;

带包编译： javac -d . 源文件名.java
带包运行： java 包名.类名
编译UTF-8文件中的中文方式： javac -encoding UTF-8 源文件名.java
javadoc ‐d 指定文档的生成位置（一般用. 代表在当前目录下生成文档） 源文件名.java

四、编码规范

Java敏感区分大小写
1. 类名，每个首字母大写
2. 包名，全部字母小写
3. 变量与函数，驼峰式（第二个单词开始首字母大写）
4. 常量， 字母全部大写，单词以_连接

五、变量

是计算机内存中的一块存储空间，是数据存储的最基本单元。

变量定义

  1. 变量的组成： 数据类型 + 变量名 + 数据（值）

  2. 变量的使用： 先声明、再赋值、后使用

    1> 声明变量
    数据类型 变量名；
    int a;
2> 赋值
    变量名 = 值；
    a = 10;
3> 使用变量
    System.out.println(a);    // 在控制台中打印出变量a的值
    另，System.out.println("a");    // 相当于打印出字母a

  3. 变量的其他定义方式：

    1> 在声明的同时赋值
    数据类型 变量名 = 值；
    int a = 10;
2> 多个相同类型变量的定义
    数据类型 变量名1，变量名2=值，变量名3...;
    int a=10,b=5,c;

注意：

Java是强类型语言，变量的类型 必须 与数据的类型一致。

  1. 变量的赋值

    // 一个变量可以被多次赋值，但该变量的值应为最后一次被赋予的值
eg:
    int a = 10;
    a = 20;
    System.out.println(a);    // a最终的值打印为20

// 可以通过变量名，给另外一个变量赋值
eg:
    int a = 10;
    int b = a;
    System.out.println(b);    // b最终的值打印为10

// char类型变量的赋值
1> 直接用字符赋值
    char a;
    a = 'A';
2> 用整数给char类型赋值
    char a;
    a = 65;    // 整数与字符通过ASCI码进行对照
3> 用unicode码进行赋值
    char a;
    a = '\u0041';

六、逻辑运算符

1. && 逻辑（短路）与
    a）语法：布尔表达式1 && 布尔表达式2
    b) 作用：当 布尔表达式1 与 布尔表达式2 结果都为true时 整个表达式为true，否则为false
    c）特性：如果布尔表达式1为 false 则不再判断 布尔表达式2

2. & 按位与
    a）语法：布尔表达式1 & 布尔表达式2
    b）作用：当 布尔表达式1 与 布尔表达式2 结果都为true时 整个表达式为true，否则为false
    c）特性：两个布尔表达式无论true 或 false 都会被判断

3. || 逻辑（短路）或
    a）语法：布尔表达式1 || 布尔表达式2
    b）作用: 布尔表达式1 与 布尔表达式2 有一个结果true时 整个表达式为true，否则为false
    c）特性：如果布尔表达式1为 true 则不再判断 布尔表达式2

4. | 按位或
    a）语法：布尔表达式1 | 布尔表达式2
    b）作用: 布尔表达式1 与 布尔表达式2 有一个结果true时 整个表达式为true，否则为false
    c）特性：无论两个布尔表达式的结果是否为true 均需要全部判断

5. ！对表达式结果取反
    a）语法：！（布尔表达式）
    b) 作用：将（）内的布尔表达式结果取反
